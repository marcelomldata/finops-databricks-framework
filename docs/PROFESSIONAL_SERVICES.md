## FAQ

### O framework open source é suficiente?

Para muitas organizações, sim. O framework open source fornece todas as funcionalidades básicas de assessment e revalidação. Professional Services são recomendados quando você precisa de:
- Automação completa
- Integração com billing cloud
- Suporte especializado
- Aceleração de implementação

### Posso usar apenas partes dos serviços?

Sim! Nossos serviços são modulares. Você pode contratar apenas o que precisa.

### Quanto tempo leva para ver resultados?

Depende do serviço:
- Implementação completa: 2-4 semanas
- Automação: 3-6 semanas
- Playbooks de correção: 4-8 semanas
- Resultados financeiros: Geralmente visíveis no primeiro mês

### Vocês oferecem suporte para o framework open source?

Oferecemos suporte prioritário através de Professional Services. Para questões gerais, use as issues do GitHub.

### Os serviços incluem treinamento?

Sim! Todos os serviços incluem treinamento da equipe para garantir que você possa manter e evoluir a solução.
